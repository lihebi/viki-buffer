# mac上添加ISU Cymail

## 网页端设置
在google设置里面，`Forwarding and POP/IMAP`中，打开`Enable IMAP`选项。
## Mac上
添加`其他账户`,填写`Full Name`, `Email`, `Password`。
### Incoming Mail Server
配置`IMAP地址`为`imap.gmail.com`，用户名密码照样填写。
#### Outgoing Mail Server
`smtp.gmail.com`。用户名密码照样。
