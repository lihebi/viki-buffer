# Use custom Directive
## Element
index.html
''<product-title></product-title>

app.js
''app.directive('productTitle', function() {
''  return {
''    restrict: 'E', // Element
''    templateUrl: 'product-title.html'
''  };
''});

## Attribute

''app.directive('productTitle', function() {
''  return {
''    restrict: 'A', // Attribute

''});
